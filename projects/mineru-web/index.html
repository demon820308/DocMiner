💡 快速入门

                                    
DocMiner 是一款基于高性能 MinerU 的智能文档解析桌面客户端。它能够将 PDF 文档及 JPG/PNG 图片深度解析为高质量的 Markdown 及结构化 JSON 格式，适用于 RAG 知识库建设、大模型语料清洗及学术文档数字化。

                                    

                                        
1 拖拽文件至解析主页，或点击选择 PDF、PNG、JPG 文件。

                                        
2 系统在后台调用高性能解析引擎进行布局分析、公式与表格提取。

                                        
3 自动完成解析，双栏对照预览解析结果，一键保存 Markdown、导出 PDF 或查看 JSON 数据。

                                    

                                


                                

                                    
⚙️ 解析引擎介绍

                                    

                                        Hybrid (混合引擎)：推荐首选。结合文本提取与 OCR（光学字符识别），自动检测文档中的文本质量并进行混合处理，解析速度与精度达到极佳平衡。
                                    

                                    

                                        VLM (多模态大模型引擎)：利用多模态视觉大模型对版面进行深度理解，适用于排版极其复杂、多栏混排或具有大量图表插图的非标准文档。
                                    

                                    

                                        Pipeline (传统分析管线)：标准的传统版面分析管道。依次进行 OCR、版面分析和公式/表格检测，适用于标准扫描件或低解析度图像。
                                    

                                


                                

                                    
🛠️ 核心功能与参数说明

                                    

                                        

                                            公式识别
                                            自动识别文档中的行内公式与块公式，并精准转换为标准的 LaTeX 代码（通过 KaTeX 渲染展示）。
                                        

                                        

                                            表格识别
                                            将文档中的有线、无线、复杂网格表格识别并转换为结构化的 Markdown 表格或 HTML 数据。
                                        

                                        

                                            强制 OCR
                                            无视 PDF 自带的文本层，全部进行光学字符扫描，适合解决 PDF 文本层字符乱码、字体编码受损或文字错位的问题。
                                        

                                        

                                            文档语言
                                            设置 OCR 引擎 of 语言偏好，支持自动检测、中文、英文。准确的语言设置能显著提升文字提取精度。
                                        

                                        

                                            下载源与缓存
                                            模型下载源支持 HuggingFace 和 ModelScope。国内网络推荐使用 ModelScope。支持自定义模型缓存目录，以便释放 C 盘空间。
                                        

                                    

                                


                                

                                    
🖨️ 导出与 PDF 打印预览说明

                                    
在解析结果对比页面的右侧，系统提供了两个主要的导出选项：保存 MD（保存 Markdown 源文件）和 导出 PDF。点击“导出 PDF”后将自动进入专业的 PDF 打印预览页面，主要操作与配置如下：

                                    
    
                                        
  • 标准排版预设：
                                            
      
                                                
    • 标准原版：采用标准 A4 字号（10.5pt）与常规段落行高，适合通用文档阅读与直接保存。
      • 
                                                
    • 自适应一页 / 极简单页：通过等比例压缩字号、段落行高、块边距，使多页文档内容更加紧凑，尽可能将内容合理适配排版在最少页数中。
      • 
                                                
    • 表格微缩：保持正文字号不变，但将所有表格字号与单元格间距最小化，适合排版表格内容庞大、密集的报表类文档。
      • 
                                                
    • 自定义：开启右侧悬浮的“自定义参数调节面板”，由您完全控制页面细节。
      • 
                                            
    
                                        
    • 
                                        
  • 自定义参数调节面板：在选择自定义预设时，可在浮动面板中通过滑块手动实时调节 字号大小、标题字号、表格字号、段落行高、段落边距、单元格内衬（表格单元格 padding） 尺寸。
    • 
                                        
  • 实时页面辅助工具：
                                            
      
                                                
    • 物理标尺：页面左侧及上方提供精确的物理厘米（cm）刻度标尺，实时感知打印后的物理排版大小。
      • 
                                                
    • 安全裁切线：在预览画面 28.5cm（临近 A4 纸底端）位置显示红色虚线。该线用于提醒打印页面的物理切割界限，若有内容刚好跨越安全线，可通过微调字号或行高进行优化，防止文字在分页处被切分。
      • 
                                            
    
                                        
    • 
                                        
  • 输出操作：
                                            
      
                                                
    • 打印：调用系统原生打印机对话框进行纸张打印。
      • 
                                                
    • 另存为 PDF：将调整排版后的最佳效果以高分辨率另存为 PDF 文件，便于传阅和保存。
      • 
                                            
    
                                        
    • 
                                    

                                


                                

                                    
📋 任务与收藏管理

                                    
    
                                        
  • 历史管理：支持模糊检索历史任务的文件名；支持根据任务状态（等待中、处理中、已完成、失败）进行快速过滤。支持任务记录的删除与对应本地输出目录的一键直达。
    • 
                                        
  • 文档收藏：点击解析对比页面右上角星号（☆）即可收藏当前文档，收藏的文档将被长期保存在“我的收藏”标签内，即使清理了常规历史，亦能随时快速调阅。
